## üé® Customization Guide

### Changing Colors

Edit `css/style.css` root variables:

```css
:root {
    --color-primary: #FF6B9D;        /* Your brand pink */
    --color-secondary: #FDB833;      /* Your brand yellow */
    --color-accent-1: #4ECDC4;       /* Your brand turquoise */
    /* ... update all colors */
}
```

### Updating Contact Information

Replace all instances of:
- Phone: `+1 (555) 123-4567`
- Email: `info@kiddiesfunworld.com`
- Address: `123 Fun Street, Playland City, ST 12345`

### Adding Real Images

1. Create `images/` directory
2. Replace `.image-placeholder` divs with `<img>` tags
3. Update references in HTML
4. Add `lazy` class for lazy loading

### Modifying Pricing

Edit pricing sections in:
- `pricing.html`: Entry fees, memberships
- `parties.html`: Party package prices
- `programs.html`: Workshop and camp fees

### Changing Business Name

Find and replace "Kiddies Fun World" throughout all HTML files.
